To run the automation script:
1. Press Win + X to open the Windows menu.
2. Select -
   Windows 10: Windows Powershell (Admin)
   WIndows 11: Windows Terminal (Admin)

3. Navigate to directory you've downloaded this bundle of scripts from using in the terminal.
4. Type:
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   .\master.ps1

5. Follow instructions on wizard.
